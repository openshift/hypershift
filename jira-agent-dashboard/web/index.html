Dashboard Overview

This dashboard tracks the HyperShift JIRA Agent — an automated system that picks up Jira issues, generates code fixes with Claude, opens pull requests, and responds to reviewer feedback. It scrapes Prow CI artifacts and GitHub to surface issue outcomes, cost breakdowns, merge timelines, and review comment analysis across every agent run.
